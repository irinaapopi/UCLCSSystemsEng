Prototyping Tools

Prototype Tool Pros Cons
PowerPoint Slideshow
  • Easy to use
  • Can be created in a short amount time
  • Does not have complex graphs
  • Restricted to pre-defined steps
Processing
  • Can produce complex interactive graphs
  • Takes a long time to create prototypes
D3.js
  • Can produce complex graphs in web browser
  • Steep learning curve
Chart.js
  • Can produce complex graphs in web browser
  • Easy to learn
  • Can easily make changes
  • Takes reletively long to create prototype
BotSociety
  • ChatBot prototypes can be created quickly
  • Easy to create prototypes
  • Hard to make large changes
We concluded that we were going to use Chart.js and BotSociety to create the prototype.
